# Galería de Imágenes Interactiva

Aplicación web interactiva desarrollada con:

- HTML5
- Bootstrap 5
- JavaScript

## Funcionalidades

- Agregar imágenes mediante URL
- Selección única de imágenes
- Eliminación de imagen seleccionada
- Diseño responsive con Bootstrap
- Atajos de teclado:
  - Enter → Agregar imagen
  - Delete → Eliminar imagen

## Conceptos aplicados

- Manipulación del DOM
- Eventos (`click`, `keydown`)
- `createElement`
- Clases dinámicas
